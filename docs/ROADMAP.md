# Дорожная карта Article Bot

## Направление проекта

Личный бот для чтения статей из Telegram. Проект развивается от простого чтения ссылок к Docker-first Python-сервису с SQLite, OpenAI-совместимым LLM-анализом, будущим RAG по Obsidian/GitHub и Codex automation как периодическим интеллектуальным ревьюером. VK рассматривается как будущий дополнительный канал, но не входит в MVP.

Предпочтительное направление:

- Основной сервис: Python.
- Первый канал: Telegram.
- VK: отложить до отдельного будущего adapter.
- Запуск: Docker Compose с самого начала.
- Хранилище: файл SQLite в постоянной папке `data/`.
- LLM: OpenAI-совместимое API через переменные окружения.
- Извлечение текста статей: Python `trafilatura`.
- Будущий RAG: Python-интерфейсы для chunking, embeddings и поиска; SQLite остаётся source of truth.
